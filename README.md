## Aquality Tool 1.0v

# Key Features
```console
**change_info()** <- Change the server's name and image using **guild.edit()**, you absolutely must have the .png image so it doesn't throw an error
**create_channels()** <- Creates 25 text channels and 25 voice channels.
**dmspam()** <- sends messages to all users on the server every 10 seconds
**spam()** <- It sends the same message five times to each server created

**main()** <- Main function: combines all the parts into one (Option 1 Ñ

```
